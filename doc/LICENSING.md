# Licensing FAQ

This project uses the **Locus Community License v1.0**. The short version is:

- **Individuals** (solo developers) can use the software for any purpose, including closed source and commercial use.
- **Enterprises** (USD 250k+ annual revenue) can use it for free **only** if the portion of their product that includes or links to this package is open-sourced under an OSI-approved license.
- Enterprises that want to keep that portion closed-source must obtain a commercial license.

Below are common questions and answers.

## What counts as an Individual?
An Individual is a natural person acting in a personal capacity or as a solo developer.

## What counts as an Enterprise?
Any person or entity (including affiliates) with **gross annual revenue of USD 250,000 or more**.

## What does “open source the part using this package” mean?
If your product includes or links to this package, the code of that part must be made publicly available under an OSI-approved license.

Examples:
- A mobile app that links this package: the app code that links to this package must be open-sourced.
- A server/service using this package: the service code that links to this package must be open-sourced.

## Can Individuals use it in closed-source commercial apps?
Yes. Individuals can use it for any purpose, including closed-source, distribution, and monetization.

## Can Enterprises use it in closed-source products?

## Does this license apply to dependencies?
No. Dependencies keep their own licenses. See acknowledgements in the example app for third-party licenses.

## Is this license OSI-approved?
No. It is a custom license designed to allow individual use while restricting enterprise closed-source use.
